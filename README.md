# Stochastic-Risk-Modeling-Analytics 📊
### Probabilistic Disease Introduction & Sensitivity Analysis (CTMC & ODE)

## 📌 Project Overview
This repository provides a high-performance computational framework in **R** to analyze and forecast pathogen introduction risks in stressed populations. The project integrates **Continuous-Time Markov Chains (CTMC)** and **Stochastic Branching Processes** to evaluate extinction probabilities and system resilience under environmental variability.

## 🛠 Key Features & Methodology
- **Stochastic Modeling:** Implementation of CTMC to simulate first-pathogen introduction and extinction risks (`extinctions_first_introduction_CTMC.R`).
- **Sensitivity Analysis (PRCC):** Development of Global Sensitivity Analysis (Partial Rank Correlation Coefficient) to identify key drivers of system outbreaks.
- **Hybrid Simulations:** Comparative analysis between deterministic ODE models and stochastic simulations.
- **Data Visualization:** Advanced statistical plotting including Violin plots and time-series profiles for risk assessment.

## 🧪 Technical Components
- **`stress_SIR_simulations.R`:** Core engine for simulating population dynamics under environmental stress.
- **`Mean_CTMC_many_alpha.R`:** Large-scale stochastic simulations across multiple parameter scenarios.
- **`Rt_profiles_ODE.R`:** Dynamic calculation of the effective reproduction number over time.

## 💻 Tech Stack
- **Language:** R (Expert Level)
- **Mathematical Methods:** Markov Chains, Branching Processes, Sensitivity Analysis, Numerical Integration.
- **Visualization:** ggplot2, Custom statistical distributions.
